


# Détection d'Émotions avec Furhat

**Prérequis**
- Python 3.8+
- PyTorch, OpenCV, PIL, torchvision, requests
- API Furhat (furhat_remote_api)
- Serveur Ollama local actif avec le modèle mistral
- Caméra connectée



**Dossiers requis**
emotion_detection_models/4classes.pth -> modèle avec 4 classes: ['angry', 'fear', 'happy', 'sad']
emotion_detection_models/5classes.pth -> modèle avec 5 classes: ['angry','disgust', 'fear', 'happy', 'sad']
